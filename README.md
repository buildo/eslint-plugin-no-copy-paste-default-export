# no-copy-paste-default-export ESLint plugin

## Installation

`npm i eslint-plugin-no-copy-paste-default-export`

## Configuration

```
{
  "plugins": [
    "no-copy-paste-default-export"
  ]
}
```

## Rule

A single rule checking that any `export default class <name>` has `<name> === <filename>`
